E-Cuti Pengadilan adalah aplikasi yang digunakan untuk mengelola pengajuan cuti pegawai di lingkungan pengadilan secara digital. Sistem ini bertujuan untuk menggantikan proses manual menjadi lebih cepat, transparan, dan terdokumentasi dengan baik.

🎯 Tujuan
Mempermudah pengajuan cuti pegawai
Mempercepat proses persetujuan oleh atasan
Menyimpan data cuti secara terpusat
Mengurangi penggunaan kertas (paperless)
⚙️ Fitur Utama
Pengajuan cuti online
Persetujuan (approval) oleh atasan
Riwayat cuti pegawai
Notifikasi status pengajuan
Manajemen data pegawai
🛠️ Teknologi yang Digunakan

Backend: PHP / Laravel
Frontend: HTML, CSS, React
Database: MySQL / NoSQL
Tools: Git & GitHub
