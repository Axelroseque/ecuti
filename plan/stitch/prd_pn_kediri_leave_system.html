# PRD: Sistem Informasi Cuti - Pengadilan Negeri Kediri Kelas I-B ## 1. Tech Stack & Tooling - **Framework:** Vite + React 18 - **Styling:** Vanilla CSS (CSS Modules) - **Icons:** Material Symbols - **Font:** Inter (Google Fonts) - **Charts:** Chart.js or Recharts - **PDF Preview:** react-pdf - **Calendar:** Custom CSS Grid calendar - **Routing:** React Router v6 ## 2. Design System ### 2.1 Color Palette - **Primary (Maroon):** #8B1A1A (Main), #A82620 (Hover), #6B1010 (Pressed) - **Secondary (Gold):** #C8942A (Accent), #E8B44A (Hover), #A87720 (Pressed) - **Neutral Dark:** #1A1D23 (Sidebar), #2A2D35 (Card Dark), #3A3D45 (Border) - **Neutral Light:** #F5F6FA (Page BG), #FFFFFF (Card BG), #E8EAF0 (Divider) - **Status:** Success (#16A34A), Warning (#F59E0B), Danger (#DC2626), Info (#2563EB) ### 2.2 Typography - **Heading:** Inter, 600–700 weight - **Body:** Inter, 400 weight - **Mono/Data:** JetBrains Mono (for NIP, codes) ### 2.3 Spacing & Radius - **Base:** 4px unit - **Radius:** 8px (Cards), 12px (Modals), 24px (Pills) - **Shadow:** Subtle (0 1px 3px), Elevated (0 4px 16px) ## 3. Core Role-Based Navigation - **Pegawai:** Dashboard, Profil, Riwayat, Ajukan Cuti, Cuti Saya. - **Atasan:** Persetujuan Masuk, Kalender Tim, Riwayat Persetujuan. - **Kepegawaian:** Verifikasi, Kuota, Data Pegawai, Laporan. - **Pimpinan/Admin:** Persetujuan Akhir, Analitik, Workflow, User Management. ## 4. Key Page Layouts - **Login:** Glassmorphism card, maroon gradient background. - **Master Layout:** Fixed Top Bar, Collapsible Sidebar (240px to 64px), Main Content Area. - **Dashboards:** Stat cards with progress bars, recent activity, mini calendar. - **Leave Form:** 4-step wizard (Type -> Detail -> Docs -> Confirm). - **Status Tracking:** Timeline view of approval flow. - **HR/Pimpinan:** Detailed tables, verifications, and data analytics charts.
